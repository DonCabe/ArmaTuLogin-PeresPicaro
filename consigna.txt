Consigna
    Para tu primera entrega, te proponemos que crees un programa
    que permita emular el registro y almacenamiento de usuarios 
    en una base de datos. Hazlo utilizando el concepto de 
    funciones, diccionarios, bucles y condicionales.

Objetivos
    Practicar el concepto de funciones.

    Desarrollar la parte lógica para el registro de usuarios.

Requisitos
    Diccionarios (guardado de datos)       CHECK

    Input (solicitud de datos)      CHECK

    Variables       CHECK

    If (chequeo de datos)      CHECK

    While (iteración para el programa, sea para agregar, loguear
    o mostrar)      CHECK

    For (recorrer datos y para búsqueda)      CHECK

    Print      CHECK

    Funciones separadas para registro, almacenamiento y muestra
    CHECK

Recomendaciones
    El formato de registro es: Nombre de usuario y Contraseña.

    Utilizar una función para almacenar la información y otra 
    función para mostrar la información.

    Utilizar un diccionario para almacenar dicha información, 
    con el par usuario-contraseña (clave-valor).

    Utilizar otra función para el login de usuarios, comprobando
     que la contraseña coincida con el usuario.

Formato
    El proyecto debe compartirse utilizando Colab bajo el nombre
     “ArmaTuLogin+Apellido”, por ejemplo “ArmaTuLogin+PeresPicaro”
